# Output Examples

This directory contains examples of the output formats generated by the system.

## Example Files:

### CSV - Invoices
`ejemplo_facturas.csv`
```csv
invoice_id,date,vendor,client,tax_id,subtotal,tax,total,items_count
INV-2024-001,2024-03-15,TechCorp Solutions S.A. de C.V.,...
```

### JSON - Sales Report
`ejemplo_reporte_ventas.json`
```json
[
  {
    "id": "V001",
    "salesperson": "John Smith",
    "region": "North",
    "sales": 45000.00,
    ...
  }
]
```

## Google Sheets

When exporting to Google Sheets, the result looks like this:

| ID | Salesperson | Region | Sales | Commission | Date |
|----|-------------|--------|-------|------------|------|
| V001 | John Smith | North | $45,000 | $4,500 | 2024-01-05 |
| V002 | Mary Johnson | South | $38,000 | $3,800 | 2024-01-08 |
| V003 | Carlos Lopez | Central | $52,000 | $5,200 | 2024-01-12 |

The first row is automatically formatted as a header (bold, gray background).
